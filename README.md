# FireBaseBoard

### FireBase로 구현한 갤러리 게시판


- 구글계정, 애플계정, 이메일 로그인 구현.

- 계정마다 게시글(사진, 제목, 내용) 작성이 가능하며 다른 게시글에 LIKE 버튼을 줄 수 있음.

- LIKE 버튼은 firebase의 문서 트랜잭션을 참고하였으며,

- LIKE버튼을 누른 계정수 만큼 count 하도록 함.
